Le protocole n°3 est développé de façon numérique. C’est à dire que pour mettre en place cette expérimentation, j’interviens par le biais du code procédurale.

Pour parvenir à ce résultat, j’utilise le logiciel processing pour développer mon code. Dans un premier temps, le code pixelise l’image du caractère, puis il vient implémenter une variable de temps et une variable de mouvement sur l’axe z afin que le caractère translate.

Cette expérimentation transforme le caractère de façon singulière et lui donne un effet de mouvement.
