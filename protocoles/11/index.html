Le protocole n°10 est développé de façon numérique. C’est à dire que pour mettre en place cette expérimentation j’interviens par le biais d’un code procédurale.

Pour parvenir à ce résultat, j’utilise le logiciel processing pour développer mon code. Dans un premier temps, le dessin du caractère est renseigné de façon mathématique par la programmation de ses points géographiques. Puis dans un second temps, j’applique une variable de déplacement sur les axes x et y pour donner un effet d’inspiration et d’expiration.

Cette expérimentation permet au caractère de donner la sensation qu’il respire. De plus, elle permet de créer deux déclinaisons : une light et une bold.
